# Финальное исправление

Замените в существующем репозитории два файла:

- `generate.py`
- `.github/workflows/update.yml`

После отправки изменений GitHub Actions автоматически пересоберёт
`ultimate-podkop-list.txt`.

Список основан на актуальном `Russia inside` и отдельных сервисных списках
проекта `itdoginfo/allow-domains`. Искусственного лимита 5000 доменов нет.
